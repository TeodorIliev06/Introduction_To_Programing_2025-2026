Задача 1

Напишете функция, която отпечатва елементите на двумерен масив, като начинът на обхождане зависи от два параметъра:

r – обхождане по редове (rows)

c – обхождане по колони (columns)

и втори параметър:

0 – отпечатване в нормален ред

1 – отпечатване в обратен ред

Програмата първо въвежда буквата за метода на обхождане (r или c), след това числото за реда (0 или 1).
След това отпечатва всички елементи на масива според избраната комбинация.

Пример:

Вход:
``` 
Rows/Cols: r
Reverse: 1
```

Изход:
``` 
9 8 7
6 5 4
3 2 1
```

Задача 5 — Умножение на матрици

Да се напише функция за умножение на две матрици.
Максимален размер на матриците = 100x100 
Преди умножение трябва да се провери дали е възможно (първата матрица е n×m, втората — m×k).

За да получим елемент C[i][j] от резултата:

взимаме i-тия ред на матрица A

взимаме j-тата колона на матрица B

умножаваме елемент по елемент

събираме всички произведения

Тоест:

C[i][j] = A[i][0] * B[0][j] +
         A[i][1] * B[1][j] +
         A[i][2] * B[2][j] + ...

Пример

A (2×3):
```
1 2 3
4 5 6
```

B (3×2):
```
7  8
9 10
11 12
```

Резултат (2×2):
```
58  64
139 154
```

Примерно:
58 = 1*7 + 2*9 + 3*11

Задача 6 — Движение в лабиринт

```
####################
#     #        #   #
# ### # #### #   # #
# #     @    ###   #
# # ####### #### ###
# #       #        #
# ####### ### #### #
#              #               
####################
```
